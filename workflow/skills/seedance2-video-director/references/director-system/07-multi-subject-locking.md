# 07. 多主体锁定

## 解决的问题

处理多人、多物体、多素材、多角色时的串台、换脸、换衣、比例漂移和动作互相污染。

## 写作检查

- 有几个主体，每个主体的 `@素材名` 是什么。
- 每个主体的位置是否固定：左/右、前/后、近/远。
- 谁在当前时间段动作，谁保持不动。
- 哪些外观必须保持：服装、发型、年龄、道具、产品结构。
- 哪些道具不能漂移或改变：桌面物、手中物、背景关键物。
- 是否需要分区、分时、分职责，避免一句话塞多个同步动作。

## Seedance 编译要点

先写主体表，再写时间线。每个 `@素材名` 只负责明确对象，不要让多个参考图争夺同一主体。

```text
@父亲设定图 始终在画面右后方，@男孩设定图 始终在左前景。0-3 秒两人不动，只让手机震动；3-6 秒只让男孩抬头，父亲保持站姿；6-8 秒父亲把本子推近，男孩手不离桌面。
```

## 常见坏写法

```text
一家三口都很开心地在厨房做饭，孩子跑来跑去，爸爸妈妈互动自然。
```

坏在多人同步动作过多，位置和职责不清。

## 失败修正

- 如果角色互换：写“始终在左/右/前/后”，并禁止交换服装和站位。
- 如果多素材污染：限制每张图只约束一个主体或一个层面。
- 如果道具漂移：明确道具位置和不移动时间段。
- 如果动作互相串台：分时写“只让 A 动，B 保持不变”。

## 最小示例

```text
@女孩设定图 坐在左侧，@男孩设定图 站在右侧，桌上杯子不移动。0-3 秒两人静止；3-5 秒只让女孩拿起杯子；5-7 秒男孩只后退半步。禁止两人交换位置、服装和手中道具。
```
