# 04. 镜头定距

## 解决的问题

用景别、焦段、机位和观看距离控制观众心理距离。镜头不是“好看视角”，而是观众站在哪里看。

## 写作检查

- 这个镜头要让观众看见信息、感到处境，还是进入情绪。
- 景别是否匹配任务：远景交代空间，中景讲关系，近景看状态，特写看证据。
- 机位是否有理由：低机位增加压迫/力量，高机位表达脆弱/被观察。
- 焦段是否会破坏比例：广角不适合近距离人脸，长焦适合压缩空间和情绪。
- 前后镜头是否递进，不要同一秒同时要求全景和眼神特写。
- 竖屏构图是否给主体留出安全空间。

## Seedance 编译要点

每个时间段只指定一个主要景别和一个情绪任务。

```text
0-3 秒远景交代 @学校场景图 的楼道长度；3-6 秒中景拍孩子站在门口；6-8 秒手部特写拍他松开衣角。
```

## 常见坏写法

```text
特写镜头展示人物全身和巨大背景，远景突出眼神细节。
```

坏在景别互相冲突。

## 失败修正

- 如果信息看不清：把远景改中景或细节特写。
- 如果情绪不成立：把镜头靠近，并减少环境信息。
- 如果人物比例变形：避免近距离广角怼脸，改用自然中近景。
- 如果剪辑跳跃：写清从远到近或从近到远的镜头递进。

## 最小示例

```text
0-2 秒中景，父亲站在门边，孩子在画面左前方。2-5 秒镜头切手部特写，孩子攥着纸角。5-7 秒回到中近景，父亲只看向孩子不说话。镜头距离克制，不要怼脸。
```
