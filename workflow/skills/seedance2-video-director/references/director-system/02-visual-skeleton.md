# 02. 画面拆骨

## 解决的问题

把抽象风格拆成可见空间结构，避免 Seedance 把所有元素贴成一张平面海报。画面必须有主体、环境、构图、前景、中景、背景和纵深。

## 写作检查

- 主体是谁，位于画面哪个区域，占画面多大。
- 环境在哪里，时间、天气、空间功能是什么。
- 前景是否有遮挡或引导物：门框、雨滴、玻璃、桌沿、植物。
- 中景发生什么动作，主体和关键道具相距多远。
- 背景提供什么信息：纵深、情绪、危险点、人物关系或品牌气质。
- 是否需要固定空间关系，防止镜头切换后方向错乱。

## Seedance 编译要点

把“风格”写成空间层次。使用图片参考时，不要只写“参考 @图”，要说明它约束哪一层。

```text
@客厅场景图 只作为空间参考：前景是沙发扶手，中景是茶几和站着的父亲，背景是半开的厨房门。男孩位于画面左下三分之一，人物高度约占画面 45%。
```

## 常见坏写法

```text
赛博朋克城市，高级，细节丰富，人物很孤独。
```

坏在没有主体位置、前中后景和空间距离。

## 失败修正

- 如果画面挤成一团：明确前景、中景、背景各自内容。
- 如果主体太大或太小：写占画面比例或所在三分区。
- 如果场景漂移：指定 @场景图 只锁定空间关系，不让模型重设环境。
- 如果风格污染主体：明确风格只作用于光线/色彩，不改变主体外观。

## 最小示例

```text
竖屏 9:16，清晨厨房。前景右侧是虚化水杯边缘，中景是 @母亲设定图 的手把碗推向孩子，背景是窗边冷白晨光和餐桌轮廓。主体手部在画面下方三分之一，环境干净真实，禁止额外人物。
```
