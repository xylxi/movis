# 06. 动作生因

## 解决的问题

让人物或物体动作有触发、有阶段、有身体联动和结束状态。不要把动作写成无因果清单。

## 写作检查

- 起始状态是什么：低头、停住、握着物体、坐着、背对镜头。
- 触发原因是什么：声音、光线、别人动作、物体掉落、内心决定的身体反应。
- 第一反应是什么：眼神先动、手指停顿、肩膀收紧、呼吸变化。
- 身体联动是什么：头、肩、手、重心、衣物、脚步如何跟随。
- 情绪如何变化：克制、惊讶、松动、犹豫、确认。
- 结束状态是什么：停在可剪辑的动作或姿态上。

## Seedance 编译要点

用时间线写动作状态流：起始状态 -> 触发 -> 第一反应 -> 身体联动 -> 结果。短片段不要塞太多动作。

```text
0-2 秒女孩低头整理书页；2 秒门口脚步声响起，她手指停在书页边缘；3-5 秒先抬眼，再慢慢转头；5-7 秒肩膀放松，嘴角只轻微上扬。
```

## 常见坏写法

```text
女孩惊讶地看向镜头，然后开心笑，表情自然。
```

坏在没有触发原因和情绪过渡。

## 失败修正

- 如果表演假：补触发事件，不要直接命令表情。
- 如果动作卡顿：拆成 2-3 个阶段，并减少同步动作。
- 如果人物突然移动：写清重心、脚步、手臂和衣物反馈。
- 如果手部/物体融合：写清手指如何接触物体和物体重量。

## 最小示例

```text
男孩原本站在门口低头。听到屋里杯子轻响后，他先停住手指，抬眼看向门缝，肩膀微微收紧；两秒后他吸气，右手松开衣角，向前迈半步，停在开口前。
```
