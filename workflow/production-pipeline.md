# 制作流程

## 总流程

当前闸门：小说母本完成第一轮完整章节草稿前，不进入短剧、角色图、场景图、AI 视频和投流执行。下面流程只作为后续路线记录。

```text
小说母本
  -> 章节写作卡
  -> 小说正文
  -> 短剧改编脚本
  -> 角色图/场景图
  -> 分镜表
  -> AI 视频提示词
  -> 成片
  -> 抖音发布
  -> 数据复盘
  -> 回写小说和改编策略
```

## 小说到短剧的转换规则

只有在小说母本第一轮完整章节草稿完成后，才开始转换。

每章小说改编短剧时，优先抽取：

- 一个明确选择点
- 一个可视化道具
- 一个家庭日常场景
- 两到三个未来分支画面
- 一个父母行动示范
- 一个选择后的真实代价
- 一个章尾钩子

## 短剧测试优先章节

以下只是候选，不代表当前立即制作。

第一轮适合测试的章节：

1. 第一章：玻璃裂开的那一秒
2. 初中：错误朋友与合群压力
3. 工作：捷径诱惑
4. 生小孩：第九条路

原因：这些节点冲突明确、画面容易表达、选择代价清楚，适合转成短剧和 AI 视频。
