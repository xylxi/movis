# 视频生成交接

## 目标

把分镜整理成用户可复制到 AI 视频平台的生成清单。Seedance 任务同时使用 `seedance2-video-director`。

## 正式段落字段

```text
段落编号：
覆盖分镜：
目标时长：
导出文件名：
视频延长输入：
图片材料：
段落目的：
动作节拍：
分镜脚本：
视频提示词：
剪辑建议：
禁止项：
```

## 图片材料格式

```text
- @中文素材名，path: assets/中文素材名.png，role: reference_image，用途：...
```

## 视频延长判断

只在同时满足以下条件时使用：

- 同一空间。
- 同一时间或明确连续时间。
- 上一段末帧人物站位、动作或镜头方向需要被下一段继承。
- 平台支持上传上一段视频作为延长参考。

不要在以下情况使用：

- 换场景。
- 换日期或明显跳时。
- 进入主观闪回、未来分支、梦境或蒙太奇。
- 剪辑上本来需要硬切、声画转场或情绪断裂。

## 提示词写法

使用延长时开头必须写：

```text
使用视频延长功能，将 @上一段视频 继续延长，新增 X 秒；开头承接上一段结尾的动作、人物站位、光线和空间方向。
```

独立生成时写清：

```text
独立生成；从上一段通过剪辑硬切/声音转场进入。
```
