# 故事到单集

## 目标

把小说、真实案例、长故事或大纲拆成可拍的单集短剧。单集只服务一个主选择或一个主冲突。

## 拆解表

```text
母本来源：
正典事实：
本集主选择：
选择前压力：
可视化道具：
关键场景：
人物关系变化：
选择后的代价：
结尾钩子：
不可改动事实：
```

## 单集结构

| 组件 | 作用 |
| --- | --- |
| 开场压力 | 5 秒内让观众知道问题正在发生 |
| 主角卡点 | 让主角必须选择，不能旁观 |
| 错误路径 | 显示逃避、误解或诱惑的代价 |
| 行动选择 | 主角做出可拍动作 |
| 现实代价 | 选择后有损失或责任 |
| 未完成动作 | 给下一集或评论区留下问题 |

## 注意

- 不把整章信息塞进一集；优先保留能驱动画面的因果链。
- 角色成长通过动作顺序呈现，不靠金句总结。
- 每集只引入必要人物和必要场景；可合并次要角色。
