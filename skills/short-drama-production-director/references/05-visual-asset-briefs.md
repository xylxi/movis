# 视觉资产 Brief

## 目标

把剧本中的人物、场景和道具整理成可生图、可复用、可进入镜头节拍和视频交接的资产规格。

## 视觉入口清单

```text
人物资产：
场景资产：
道具资产：
关键动作：
需要一致性的外观：
可以只在镜头节拍里描述的元素：
不需要生图的元素：
```

## 人物 Brief


## 场景 Brief

```text
@中文素材名：
场景功能：
空间关系：
入口/出口：
关键道具：
可拍动作点：
光线和时间：
禁用元素：
```

## 原则

- 人物设定图用于锁定形态；单张角色照只作为补充。
- 场景图用于锁定空间关系；单张氛围图不能替代可拍空间。
- 给视频模型的场景参考图必须是单场景、无标记线、无多格拼版。
