# 分镜

## 目标

把剧本转成镜头、动作节拍和素材引用。分镜服务视频生成，不是文学复述。

## 分镜字段

```text
分镜编号：
场次：
时长：
引用素材：
画面：
动作：
镜头：
声音：
转场：
禁止项：
```

## 素材引用

每条分镜的引用素材使用中文名：

```text
@中文素材名，path: assets/中文素材名.png，role: reference_image，用途：...
```

规则：

- `@素材名` 和文件名必须一致。
- 不使用机器名或深层目录作为交付路径。
- 同一素材在分镜、视频 handoff 和实际文件中保持同名。
- 场景参考图只对应一个场景或空间功能。

## 导演节拍和正式生成段

- 分镜可以细到每个镜头。
- 正式视频生成可以把多个连续分镜合并成一个段落。
- 段内导演节拍说明该段内部时间线，不等于每个节拍都单独生成。
- 合并时优先保持同一空间、同一人物状态和同一动作链。
