# 剧本写作

## 剧本格式

```text
标题：
时长：
场次：
人物：
道具：

场次 1 / 地点 / 时间
画面：
动作：
对白：
声音：
转场：
```

## 写法

- 先写可拍动作，再写对白。
- 每场只保留一个明确功能：建立压力、推进选择、呈现代价或收束钩子。
- 对白服务关系和行动，不替作者解释主题。
- 重要信息通过道具、手机消息、账单、空间变化、人物停顿等画面承载。
- 轻科幻、悬疑或反转设定只在关键选择点出现，不把规则解释铺满全片。

## 输出检查

```text
开场是否有外部压力：
主角是否必须行动：
每场是否能拍：
道具是否自然出现：
对白是否过度解释：
结尾是否有未完成动作：
```
