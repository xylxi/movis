# 章节写作卡模板

每章开写前先填这张卡。写完后按同一张卡自检。

```text
章节名：
年龄/阶段：

现实事件：
这一章必须发生在一个具体事件里，不能只写感悟。

家庭背景切入：
通过钱、时间、空间、家务、工作、维修费、接送、饭桌等细节进入，不要大段说明。

父亲当下压力：
他作为普通职场员工，这一章面对什么现实压力？

母亲当下压力：
她作为全职主妇，这一章面对什么生活/家庭/情绪压力？

主角选择：
许行现在必须做什么选择？

未来分支：
他看见哪些未来？每条未来的诱惑和代价是什么？

父母行动示范：
父母不用讲大道理，他们具体做了什么影响许行？

主角付出的代价：
他做出选择后失去了什么、承担了什么？

本章学到的能力：
勇敢、乐观、面对、复盘、承担、识别风险、沟通等。
这是作者用字段，不允许在正文中直接写成教育总结或章尾金句。

下一章种子：
本章留下什么影响，推动下一阶段？
```

## 自检问题

- 这一章能不能被一句现实事件概括？
- 如果删掉未来分支，现实冲突还成立吗？
- 如果删掉父母台词，父母的影响还能从行动看出来吗？
- 许行有没有为自己的选择付出成本？
- 章尾有没有余味，而不是硬讲主题？
- “本章学到的能力”有没有通过行动、代价和复盘场景呈现，而不是由角色说出来？
