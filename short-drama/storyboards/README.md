# 分镜与图生视频包

本目录用于整理短剧分镜表、关键帧生成提示词和 AI 视频提示词。

## 工作顺序

```text
approved 人物/场景资产
-> 分镜拆分
-> planned 首帧关键帧路径和提示词
-> image2 / image_gen 生成首帧图
-> 人工确认首帧图为 approved
-> 即梦 / Seedance 图生视频提示词
-> 视频生成与复盘
```

## 规则

- 不直接把人物设定图或场景设定图拼贴图作为图生视频输入图。
- 每个分镜必须先有首帧图路径和关键帧提示词。
- 每个分镜必须引用 `approved` 人物和场景资产。
- 图生视频提示词只描述镜头运动、人物动作、光线和情绪，不新增正典事实。
- 禁止系统 UI、弹窗、任务奖励、玄幻命定视觉和狗血强冲突。
