# 分镜与视频生成包

本目录用于整理短剧分镜表、关键帧生成提示词和 AI 视频提示词。正式交付以分镜和视频提示词为主，首帧关键帧图是提高一致性的可选增强步骤。

## 工作顺序

```text
approved 人物/场景资产
-> 分镜拆分
-> Seedance 分镜提示词交付
-> 用户在即梦 / Seedance 逐条生成视频片段
-> 下载片段并按文件名导入剪映
-> 剪映组装、配音、音效、字幕和复盘
```

高一致性增强流程：

```text
分镜拆分
-> planned 首帧关键帧路径和提示词
-> image2 / image_gen 生成首帧图
-> 人工确认首帧图为 approved
-> Seedance 图生视频
```

## 规则

- 不直接把人物设定图或场景设定图拼贴图作为图生视频输入图。
- 每个分镜必须有可复制的视频生成提示词、建议时长和导出文件名。
- 如果走图生视频，每个分镜必须先有首帧图路径和关键帧提示词。
- 每个分镜必须引用 `approved` 人物和场景资产。
- 视频提示词只描述镜头运动、人物动作、光线和情绪，不新增正典事实。
- 禁止系统 UI、弹窗、任务奖励、玄幻命定视觉和狗血强冲突。
