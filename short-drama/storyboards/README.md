# 分镜与视频生成包

本目录用于整理短剧分镜表、关键帧生成提示词和 AI 视频提示词。正式交付以分镜和视频提示词为主，首帧关键帧图是提高一致性的可选增强步骤。

## 工作顺序

```text
approved 人物/场景资产
-> 分镜拆分
-> Seedance 生成段落提示词交付
-> 用户在即梦 / Seedance 按段落生成视频片段
-> 下载片段并按文件名导入剪映
-> 剪映组装、配音、音效、字幕和复盘
```

高一致性增强流程：

```text
分镜拆分
-> planned 首帧关键帧路径和提示词
-> image2 / image_gen 生成首帧图
-> 人工确认首帧图为 approved
-> Seedance 图生视频
```

## 规则

- 不直接把人物设定图或场景设定图拼贴图作为图生视频输入图。
- 正式 Seedance handoff 应优先按连续生成段落交付，不默认把所有分镜拆成独立视频任务。
- 每个正式生成段落必须按 Seedance API 对接口径写成 15s 段落包，包含图片材料清单、图片 URL 占位符、图片 role/用途、导演脚本、可复制的视频生成提示词、导出文件名和覆盖分镜范围。
- 段内导演节拍必须写清：镜头目的、镜别/机位、镜头运动、人物调度、动作节拍、表演要求、声音设计、剪辑点、Seedance 提示词和禁止项。
- 如果走图生视频，每个分镜必须先有首帧图路径和关键帧提示词。
- 每个分镜必须引用 `approved` 人物和场景资产。
- 视频提示词必须服务镜头调度，只描述可拍动作、光线、声音和剪辑点，不新增正典事实。
- 禁止系统 UI、弹窗、任务奖励、玄幻命定视觉和狗血强冲突。
