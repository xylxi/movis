# 第一集 Seedance 2.0 视频生成交付清单：玻璃裂开的那一秒

## 用法

本文件给用户在即梦 / Seedance 2.0 中生成短视频片段使用。Codex 不在这一阶段生成首帧图或视频，用户生成片段后用剪映按文件名顺序组装成完整视频。

正式生成时不要把 24 个分镜都当成独立视频任务。24 个分镜用于导演节拍控制；实际推荐合并为 8 个连续生成段落，这样人物、光线、空间和动作衔接更稳定。

- 模型：`doubao-seedance-2-0`
- 模型页：https://console.volcengine.com/ark/region:ark+cn-beijing/model/detail?Id=doubao-seedance-2-0
- API 参考：https://www.volcengine.com/docs/82379/1520757
- 教程入口：https://www.volcengine.com/docs/82379/1541523
- 默认方式：文生视频 / 分镜提示词生视频
- 可选增强：人物或场景一致性不足时，再按分镜补首帧图，改走图生视频
- 画幅：竖屏 9:16
- 风格：现实主义短剧，2020s 当代普通小学和年轻三口之家，清晰明亮，不显老旧
- 后期：优先生成 8 个段落 MP4，下载后按文件名导入剪映，统一配音、音效、字幕和转场

## Seedance 2.0 API 对接规则

Seedance 2.0 的任务输入应按“文本 + 图片材料”组织。本文档里的本地图片路径不能直接提交给 API，调用前需要先上传为可访问的图片 URL，再替换下方占位符。

推荐每个 15s 段落按以下逻辑组装：

```json
{
  "model": "doubao-seedance-2-0",
  "duration": 15,
  "ratio": "9:16",
  "content": [
    {
      "type": "text",
      "text": "粘贴本段导演脚本"
    },
    {
      "type": "image_url",
      "role": "reference_image",
      "image_url": {
        "url": "替换为素材图片 URL"
      }
    }
  ]
}
```

图片材料使用原则：

- 角色设定图和场景设定图默认作为 `reference_image`，用于锁定人物、服装、空间和道具，不直接当首帧。
- 如果后续生成了单镜头首帧关键帧，再把关键帧作为 `first_frame` 使用。
- 单个 15s 段落不要堆太多参考图，优先选本段最关键的 2-5 张。
- 导演脚本里要明确“图片材料如何被使用”，例如“以许行设定图锁定孩子外观，以操场场景图锁定教学楼和公告栏空间关系”。
- 字幕、旁白、对白成品优先在剪映统一制作，不要求 Seedance 在画面里生成文字。

## 导演版全局规则

每条分镜提示词都按导演执行口径写，不只描述剧情。复制到 Seedance 时，建议把全局前缀、单条 `Seedance 提示词` 和全局禁止项拼在一起。

全局前缀：

```text
现实主义短剧，2020s 中国普通小学和年轻三口之家，竖屏 9:16，清晰明亮，生活流镜头，人物年龄自然年轻，不显老不疲惫，普通家庭质感，克制表演，真实空间关系。
```

全局禁止项：

```text
不要字幕，不要水印，不要可读大段文字，不要系统 UI，不要弹窗，不要玄幻光效，不要血腥，不要夸张灾难片，不要豪宅，不要精英学校，不要苦情旧屋，不要父母训斥孩子，不要鸡娃对抗。
```

## 推荐生成策略

默认使用 **8 段生成**：

```text
24 个导演分镜节拍
-> 合并成 8 个 Seedance 生成段落
-> 每段内部保留 2-4 个动作节拍
-> 下载 8 个 MP4
-> 剪映按段落顺序硬切和微调
```

为什么不默认 24 段逐条生成：

- 24 段逐条生成时，同一人物的脸、服装、光线和空间更容易漂移。
- 过短片段会让动作没完成，剪映拼接时容易出现跳轴、跳姿态和情绪断裂。
- 8 段生成能让同一空间内的动作连续发生，后期只需要裁剪节奏。

24 个分镜仍保留在后文，作用是约束每段里的镜头节拍，以及某一段失败时进行单镜头重生。

## 剪映组装顺序

按 8 个正式生成段落导入剪映：

```text
ep01_A01_danger_setup_15s.mp4
ep01_A02_choice_pressure_15s.mp4
ep01_A03_future_branches_15s.mp4
ep01_A04_reality_choice_15s.mp4
ep01_A05_school_handling_15s.mp4
ep01_A06_cost_apology_15s.mp4
ep01_A07_evening_review_15s.mp4
ep01_A08_morning_bill_15s.mp4
```

建议后期规则：

- 先只做硬切，不急着加转场，检查情绪是否顺。
- 未来分支段 `ep01_A03_future_branches_15s.mp4` 可在剪映里轻微降环境声、加冷暖色温变化。
- 如果某个 8 段生成结果只有局部不可用，先按后文 24 个段内导演节拍重生对应单镜头，再在剪映里替换局部，不要整段重做。
- 对白、旁白、字幕优先在剪映统一处理，不让模型在画面里生成字幕。
- 如果 Seedance 生成时长只能选择固定档位，先生成较长版本，再在剪映里裁到本表建议时长。
- 下载结果后尽快本地保存；视频生成任务返回的视频链接通常有有效期。
