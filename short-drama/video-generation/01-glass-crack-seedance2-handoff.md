# 第一集 Seedance 2.0 视频生成交付清单：玻璃裂开的那一秒

## 用法

本文件给用户在即梦 / Seedance 2.0 中逐条生成短视频片段使用。Codex 不在这一阶段生成首帧图或视频，用户生成片段后用剪映按文件名顺序组装成完整视频。

- 模型：`doubao-seedance-2-0`
- 模型页：https://console.volcengine.com/ark/region:ark+cn-beijing/model/detail?Id=doubao-seedance-2-0
- API 参考：https://www.volcengine.com/docs/82379/1520757
- 教程入口：https://www.volcengine.com/docs/82379/1541523
- 默认方式：文生视频 / 分镜提示词生视频
- 可选增强：人物或场景一致性不足时，再按分镜补首帧图，改走图生视频
- 画幅：竖屏 9:16
- 风格：现实主义短剧，2020s 当代普通小学和年轻三口之家，清晰明亮，不显老旧
- 后期：所有 MP4 下载后按文件名导入剪映，统一配音、音效、字幕和转场

## 全局提示词前缀

每条分镜提示词前建议加上：

```text
现实主义短剧，2020s 中国普通小学和年轻三口之家，竖屏 9:16，清晰明亮，生活流镜头，人物年龄自然年轻，不显老不疲惫，普通家庭质感，克制表演，真实空间关系。
```

## 全局禁止项

每条分镜末尾建议加上：

```text
不要字幕，不要水印，不要可读大段文字，不要系统 UI，不要弹窗，不要玄幻光效，不要血腥，不要夸张灾难片，不要豪宅，不要精英学校，不要苦情旧屋，不要父母训斥孩子，不要鸡娃对抗。
```
