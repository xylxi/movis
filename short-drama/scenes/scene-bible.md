# 场景视觉总控

## 总原则

场景视觉必须服务《第九条路》的现实亲情和轻科幻人生分支气质。场景应像普通二线城市学校、家庭和职场中真实存在的空间，不做玄幻、赛博、广告大片或悬浮豪宅质感。

默认优先生成功能明确的场景设定图，而不是单张氛围图。场景设定图用于锁定空间关系、人物动线、关键道具、光线色调和可拍动作点；单张场景氛围图只能作为补充。

## 场景 ID 规则

场景 ID 使用 `<episode>-<scene-slug>`：

- `ep01-school-playground-building`
- `ep01-broken-window-risk`
- `ep01-school-office-clinic`
- `ep01-family-dining-review`
- `ep01-next-morning-bill`

同一空间如果承担不同动作功能，可以拆成不同场景 ID。不要用“学校”“家里”这类模糊 ID 直接生图。

## 场景设定图版式

每张 `scene_sheet` 必须尽量包含：

- 主视角：观众第一次理解空间关系的 establishing view。
- 反向视角：用于后续分镜反打和动线确认。
- 平面关系或动线示意：人物站位、入口、出口、危险点、道具位置。
- 关键道具区：单独展示会被反复引用的物件。
- 光线/色调条：锁定白天/室内/清晨等色温和现实质感。
- 三个可拍动作点：角色可以自然完成的动作，不提前写分镜表。
- 禁用元素：不得出现的玄幻、系统、夸张伤害或正典外元素。

## 通用视觉风格

- 现实主义短剧场景设定图，半写实绘画式或真实感概念图均可。
- 背景干净但信息足够，能看清空间关系。
- 2020s 当代二线城市普通学校、普通家庭、普通办公室质感：普通但维护良好、干净明亮，不做年代剧、老单位或怀旧旧照感。
- 道具必须能被角色自然使用或看见，不能强行掉出或承担世界观解释。
- 数字道具可以出现，但正典数字以 Markdown brief 为准。

## 场景年代与人物呈现规则

场景应服务空间关系和道具调度，不重新定义角色年龄。场景设定图中的人物只作为比例、站位和动作参考，优先使用背影、侧影、半身遮挡或简化小人；如出现父母正脸，必须贴近已批准 v03 的 32-38 岁年轻家庭状态，不得画成中老年、憔悴或苦情家庭。

生图时必须区分：

- 学校可以普通、略有使用痕迹，但应是 2020s 仍在使用且维护正常的公立学校：干净窗框、明亮走廊、清楚操场线、普通公告栏。
- 家庭可以朴素、可负担，但应是年轻家庭正在生活的当代小户型：浅色墙面、干净餐桌、轻便收纳、自然晨光或饭桌暖光。
- 不要表现：老旧单位楼、灰败墙面、发黄窗帘、深色厚重老家具、破败柜子、怀旧年代剧、父母显老、全家疲惫苦情。

## 通用禁忌

- 不做玄幻、赛博、系统 UI、弹窗、数值面板。
- 不做豪宅、精英学校、广告棚拍或过度电影化空间。
- 不做老旧年代感、灰败苦情剧场景或把年轻父母画成中老年。
- 不夸张事故伤害，不渲染血腥。
- 不在场景设定图阶段生成分镜表、视频提示词或视频。
- 不把未来分支做成命定视觉，只能为后续关键帧保留感官和色温方向。

## 生图前检查

生成任何场景设定图前，必须确认：

1. 场景 ID 已存在。
2. 对应脚本场次、空间功能、人物动线和道具锚点已明确。
3. 已读取 `short-drama/visual-handoff/` 下对应入口清单。
4. 已读取 `short-drama/assets/scenes/_asset-index.md`。
5. `_asset-index.md` 中没有可复用的 `approved` 资产。
6. 本次优先生成 `scene_sheet`，不只生成氛围图。
