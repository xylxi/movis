# 极寒重生复仇爽剧项目规则

本文件只约束 `projects/frozen-rebirth-revenge/` 当前项目。项目事实以 `PROJECT.md`、`bible.md`、`episodes/`、`assets/`、`storyboards/` 和 `video-generation/` 为准，不混用其他剧本内容。

## 素材命名

- 最终图片统一放在 `assets/` 根目录。
- 镜头节拍和视频交付引用素材使用 `@中文素材名`。
- 文件名必须与 `@素材名` 去掉 `@` 后完全一致，例如 `@沈知夏设定图` 对应 `assets/沈知夏设定图.png`。
- 废稿、旧版本和未确认图不得进入正式视频交付。

## 成年女性角色设定图

沈知夏、沈晚、陈兰都是成年女性。女性角色图要有短视频第一眼吸引力，但必须保持真实、得体、可用于短剧连续性。

- 后续重生或新增女性设定图时，不使用羽绒服、厚棉服作为主视觉；厚外套会遮挡上身轮廓，影响吸睛表达。
- 沈知夏当前设定图已改为白色细肩带针织上衣、黑色高腰短裤、黑色袜裤和黑色尖头高跟鞋；不要再沿用旧羽绒服或白衬衫短裙方案作为主视觉。
- 极寒感交给场景、道具、后续视频动作或开敞的非羽绒外套处理；人物设定图优先锁定脸、发型、上身服装结构、裙装和鞋袜。
- `@角色设定图` 是长期一致性母图，应该在一张图里包含同一人物的正面、三分之四、侧面和必要背面/服装细节，防止后续侧脸、转身和背影变形。
- 设定母图可以多视角，但必须是同一人物、同一服装、同一光线体系，不要文字标签、箭头、参数表和复杂说明。
- Seedance 正式生视频时不要把多视角母图当唯一参考硬喂；应按镜头方向另拆或另生单视角视频参考图，例如 `@沈知夏正面参考图`、`@沈知夏侧面参考图`、`@沈知夏背面参考图`。
- 如果某段必须使用设定母图作为参考，提示词要写清“只用于锁定人物脸、发型和服装，不复现多格版式，不生成多人”。
- 女性角色之间不能撞衫。沈知夏走白色细肩带针织上衣、黑色高腰西装短裤、黑色袜裤、黑色尖头高跟鞋的冷感高智路线；沈晚走黑色拉链短袖运动上衣、灰色高腰运动短裤、白色运动鞋的甜美运动吸睛路线；陈兰走酒红衬衫、深色外套和铅笔裙的成熟控制路线。
- 沈晚的性感特征落在长发甜美脸、肩颈、锁骨、黑色贴身运动上衣、灰色高腰短裤比例和白色运动鞋上，不再使用白衬衫、黑短裙、黑长靴这一套。
- 上身吸睛感写成服装剪裁和面料表现：`低位门襟`、`前片自然结构`、`面料保持不透明`、`胸前周围有真实褶皱和轻微张力`、`整体合身但不挤压不夸张`。
- 肤色写清楚：白皙冷调、干净自然皮肤纹理、不过度磨皮。
- 年龄写实：必须明确 `成年女性` 和具体年龄；不要写未成年人感、校园制服、少女身体、童颜诱导。
- 禁止露点、透明服装、裸露性器官、性行为暗示、偷拍感、低质艳俗和过度擦边构图。

## 稳定提示词写法

不要直接堆叠空泛词，例如“高级、性感、电影感”。要把它们翻译成可见事实：

```text
成人年龄 + 身份气质
脸部状态 + 发型 + 妆容
服装颜色 + 领口/扣位/剪裁 + 面料褶皱
裙装/鞋袜 + 道具
场景背景 + 光线 + 镜头
负面约束
```

对于女性角色，优先使用这些稳定表达：

```text
成年女性，白皙冷调肤色，干净自然皮肤纹理，清透但成熟的脸部状态，
白色不透明细肩带针织上衣或角色指定上衣，领口、肩线和前片结构清楚，
上衣有真实织物纹理和自然褶皱，合身但不挤压不夸张，
高腰短裤或短裙，袜裤或长袜，尖头高跟鞋或长靴，成人都市时装，不是校服，不是未成年人感。
```

避免使用这些容易导致拒绝或跑偏的表达：

```text
裸露、挤压、夸张胸部、透明、内衣外露、少女感、学生制服、偷拍、性暗示、低俗网红。
```
